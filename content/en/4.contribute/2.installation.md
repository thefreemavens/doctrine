The Doctrine site is built using **Nuxt Content using the Docus Template**.

- [Nuxt Content](https://content.nuxt.com/)
- [Docus](https://docus.dev/en)

The following instructions will help you run your own Doctrine locally.

## Quick Start

First, clone the Doctrine repo:

::code-group

```bash [https]
git clone https://github.com/thefreemavens/doctrine.git
```

```bash [ssh]
git clone git@github.com:thefreemavens/doctrine.git
```

```bash [GitHub CLI]
gh repo clone thefreemavens/doctrine
```
::

## Setup

Make sure to install the dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
